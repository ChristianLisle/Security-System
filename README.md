# Security-System
Custom built security system

## Deploying the system:
### Pip install the following modules:
 - opencv
 - flask
 - waitress
 - pickle

## Deploy the Backend:
Run:
python3 server.py
python3 webserver.py

## Connecting client (camera):
Change HOST url (to match where server.py is deployed) and run:
python3 client.py
